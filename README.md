# Serve http via koa

### Installation

```
$ git clone https://github.com/robinmoquet/serve-resto
```

Initialisation : 
```
cd serve-resto && npm i
```

Lancement du serve de dev
```
npm run dev
```

Le projet est dispo sur l'url : 
```
http://localhost:5000/
```
